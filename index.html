info What is this?


It is very inconvient to add someone to your contacts just to send a message on Whatsapp. So why do it? (Drum rool please!) WhatsAppQuic is a WebApp that allows you to send messages to Whatsapp users without adding them to you contacts.

bubble_chart How to use


Enter the phone number of the user you wish to message in the first textbox, then enter the message you would like to send the user on the second textbox (the default is set to "Hi!")
